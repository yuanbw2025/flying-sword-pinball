## 🎮 游戏简介

**飞剑弹珠**是一款融合了中国仙侠美学的经典 Breakout（打砖块）游戏。玩家操控一把飞剑，通过反弹弹珠击碎排列的砖块，在限时内通关 8 个精心设计的关卡。

### ✨ 核心特色

- 🏯 **仙侠美学** — 暗紫深蓝的仙境配色，粒子特效满天飞，飞剑拖尾如流星
- 🎯 **8 大关卡** — 从"晨雾林"到"虚空境"，每关独特砖块布局与配色
- ⭐ **三星评价** — 每关根据用时评星，挑战极限速度
- 🏆 **排行榜系统** — 本地排行榜，记录每关最佳成绩
- 🎵 **程序化音效** — 使用 Web Audio API 实时生成音效，无需加载任何音频文件
- 📱 **全设备兼容** — 桌面键盘/鼠标、移动端触控均可操作
- ⚙️ **丰富设置** — 音效音量、音乐音量、震屏强度、粒子密度、FPS 显示均可调节
- 🎲 **趣味剑号** — 30 个随机仙侠风格昵称（"疾风剑仙""紫电候""弹珠王"…）

## 🏗️ 技术架构

### 零依赖单文件架构

整个游戏浓缩在一个 `index.html` 文件中（~43KB），包含：

```
index.html
├── <style>     CSS 样式（UI 面板 + 动画）
├── <body>      HTML 结构（菜单/设置/排行/关卡选择/游戏覆盖层）
└── <script>    JavaScript 游戏引擎（~500 行）
    ├── CONFIG          游戏常量配置
    ├── CANVAS          画布初始化与自适应
    ├── SOUND           Web Audio API 音效引擎
    │   ├── 碰撞音效    实时合成
    │   ├── 消除音效    频率扫描
    │   └── 背景音乐    程序化旋律生成
    ├── LEVELS          8 关关卡数据定义
    ├── GAME STATE      全局状态机
    ├── PHYSICS         碰撞检测 + 物理模拟
    │   ├── 球-墙碰撞
    │   ├── 球-剑碰撞（角度反射）
    │   └── 球-砖碰撞（含砖块消除）
    ├── PARTICLES       粒子特效系统
    ├── RENDER          60FPS 渲染循环
    │   ├── 背景星空
    │   ├── 砖块绘制
    │   ├── 飞剑绘制（含拖尾）
    │   ├── 弹珠绘制（含光晕）
    │   └── HUD（分数/时间/生命）
    ├── INPUT           键盘 + 鼠标 + 触控统一输入
    └── UI / SCREENS    菜单/暂停/胜利/失败画面管理
```

### 核心技术点

| 技术 | 说明 |
|------|------|
| **HTML5 Canvas** | 2D 游戏渲染，60FPS 动画循环 |
| **Web Audio API** | 所有音效和背景音乐均为程序化实时合成，零资源加载 |
| **requestAnimationFrame** | 帧同步渲染，自动适配屏幕刷新率 |
| **纯 CSS UI** | 所有菜单面板用 CSS Flex/Grid 布局 + transition 动画 |
| **localStorage** | 存储玩家设置、关卡进度、排行榜数据 |
| **响应式设计** | `clamp()` + viewport 单位，适配手机到桌面全尺寸 |

### 物理引擎

- 弹珠运动：基于速度向量 + 摩擦系数 `FRC=0.97` + 加速度 `BACC=600`
- 碰撞反射：飞剑碰撞时根据接触点位置计算反射角度，越靠边角度越大
- 砖块碰撞：AABB 碰撞检测，支持四方向精确反弹

### 粒子系统

- 砖块消除时喷射 8-16 个彩色粒子
- 弹珠碰撞时产生火花粒子
- 粒子密度可在设置中调节（低/中/高三档）
- 每帧更新粒子位置、透明度、大小衰减

## 🎨 设计理念

### 为什么是单文件？

1. **极致的可分享性** — 一个 HTML 文件可以通过任何方式分享（邮件、微信、U盘）
2. **零部署成本** — 双击就能打开，不需要服务器
3. **完全离线可用** — 无需网络连接（音效都是程序化生成）
4. **教学友好** — 一个文件就能展示完整的游戏开发流程

### 为什么用 Web Audio 合成音效？

传统方式需要加载 `.mp3` / `.wav` 文件，而本游戏的所有音效都是通过 Web Audio API 的振荡器（Oscillator）实时计算生成：
- **碰撞音**：短促的正弦波 + 频率扫描
- **消除音**：双振荡器叠加 + 指数衰减
- **背景乐**：程序化旋律生成（五声音阶 + 随机节奏）

这意味着整个游戏 **没有一个外部资源需要加载**。

## 🔧 自定义与改造

想改造游戏？以下是关键参数位置：

```javascript
// 游戏核心参数（约第 117 行）
const BR=7,          // 弹珠半径
      BACC=600,      // 弹珠加速度
      ACRT=1.8,      // 加速度倍率
      FRC=.97,       // 摩擦系数
      DSM=3.5,       // 减速倍率
      DCD=1.2,       // 减速冷却
      DMN=30;        // 最小减速
```

```javascript
// 关卡配色（约第 120 行）
const LVBG=[,
  {r:20,g:12,b:45},  // 关1 紫暗
  {r:12,g:18,b:45},  // 关2 深蓝
  {r:32,g:10,b:22},  // 关3 暗红
  // ...
];
```

想加新关卡？找到 `LEVELS` 数组，按相同格式添加砖块布局数据即可。
